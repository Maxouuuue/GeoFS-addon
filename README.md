# GeoFS-addon
L'addon "Overhead Addon-GeoFS" est un addon qui rajoute un overhead panel semi-fonctionnel (les boutons sont cliquables mais n'ont pas d'incidence sur le jeu).

L'overhead panel contient:
 - ENG 1 & 2 fire tests
 - APU fire test
 - ADIRS
 - Batteries et External Power
 - Packs 1 & 2
 - APU Master et APU Start
 - ANTI-ICE
 - External Lights
 - Cabin Signs (Seat Belts et No Smoking)
 - Cabin Intercom (Demande a la cabine si elle est prete pour le boarding, T.O, landing, deboarding (avec une probabilité de réponse négative)).
